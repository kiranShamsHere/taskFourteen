## 📖 Description
In this project, I have created different sizes of columns using Bootstrap classes, demonstrating a fundamental aspect of responsive web design. This repository showcases a simple HTML page that utilizes Bootstrap's grid system to create a variety of column layouts.

## 🎯 Features
* Utilizes Bootstrap classes to create different column sizes
* Demonstrates responsive design using Bootstrap's grid system
* Simple and easy-to-understand HTML structure

## 🌐 Tech Stack
* HTML
* CSS (via Bootstrap)

## 🗂️ Project Structure
* `README.md`: This file, containing project information and instructions
* `index.html`: The main HTML file, showcasing the column layouts
* `style.css`: CSS file for custom styling (not required for Bootstrap functionality)

## 💻 Installation
1. Clone the repository: `git clone https://github.com/kiranShamsHere/taskFourteen.git`
2. Open `index.html` in a web browser to view the project

## 📊 Usage
* Open `index.html` in a web browser to view the different column layouts
* Resize the browser window to see the responsive design in action
